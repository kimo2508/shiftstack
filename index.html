1
Download ShiftStack from this chat
Click the download icon on the ShiftStack file above. It saves as ShiftStack.html to your Downloads folder.
2
Rename it to index.html
Vercel serves the file called index.html as the homepage. Without this rename it won't load.
Right-click the file → Rename → type index.html → hit Enter
3
Create a new GitHub repo
Go to github.com → click the + in the top right → New repository. Name it shiftstack. Set it to Public. Click Create repository.
Same steps as when you created the LIV repo. You've done this before.
4
Upload index.html to the repo
On your new empty repo page, click Add file → Upload files. Drag your index.html in. Scroll down and click Commit changes.
One file only — that's the whole app
5
Import to Vercel
Go to vercel.com → Add New Project → Import Git Repository → find your shiftstack repo → click Deploy. No settings to change. Leave everything default.
When it asks about Framework Preset — leave it on Other. Do not select React, Next.js, or anything else. It's a plain HTML file.
6
Your live URL appears in ~30 seconds
Vercel gives you a URL like shiftstack.vercel.app. Open it on your phone. You'll see the onboarding screen — go through it with your real info. Set your 10-on/4-off rotation, anchor date, and pay rate.
This is your real app from now on
7
Future updates — dead simple
Whenever we update the file here in Claude: download the new version → rename to index.html → go to your GitHub repo → click the existing index.html → click the pencil/edit icon → drag the new file in → commit. Vercel auto-deploys in 30 seconds.
No terminal. No build commands. No npm. Just drag, commit, done.
